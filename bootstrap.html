📘 What is Bootstrap?

Bootstrap is a free and open-source front-end CSS framework originally developed by Twitter. It provides a collection of ready-to-use components — such as navbars, buttons, cards, grids, alerts and badges — allowing developers to build responsive, mobile-friendly websites much faster than writing CSS from scratch.

Bootstrap uses a powerful 12-column grid system powered by Flexbox, along with responsive utility classes for spacing, typography and layout. Below, we recreate three core Bootstrap-style components using only HTML & CSS.

🧭 Example 1: Bootstrap Navbar

A responsive navigation bar with a brand name and navigation links — one of the most commonly used Bootstrap components.

<nav class="navbar navbar-dark bg-primary">
  <div class="container">
    <a class="navbar-brand">Asim Portfolio</a>
    <div class="nav-links">
      <a href="#">Home</a>
      <a href="#">Projects</a>
      <a href="#">Contact</a>
    </div>
  </div>
</nav>

▶ Output:

Asim Portfolio
Home Projects Contact

🏷️ Example 2: Bootstrap Badges & Alerts

Badges are small labels used to highlight skill tags or statuses. Alerts display contextual messages to users.

<!-- Badges -->
<span class="badge bg-primary">HTML</span>
<span class="badge bg-success">CSS</span>
<span class="badge bg-warning">JavaScript</span>
<span class="badge bg-danger">React</span>
<span class="badge bg-secondary">Node.js</span>

<!-- Alerts -->
<div class="alert alert-info">ℹ️ Your profile was updated.</div>
<div class="alert alert-success">✅ Project submitted successfully!</div>
<div class="alert alert-warning">⚠️ Deadline is tomorrow.</div>

▶ Output:

HTML CSS JavaScript React Node.js
ℹ️ Your profile was updated.
✅ Project submitted successfully!
⚠️ Deadline is tomorrow.

📐 Example 3: Bootstrap Responsive Grid

Bootstrap's grid system divides a page into 12 columns. Using col-md-3, each column takes 3 of 12 columns (25%) — giving us a 4-column layout that stacks vertically on small screens.

<div class="row g-3">
  <div class="col-md-3">Frontend</div>
  <div class="col-md-3">Backend</div>
  <div class="col-md-3">Database</div>
  <div class="col-md-3">DevOps</div>
</div>

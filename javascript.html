JavaScript

The programming language that powers interactivity on the web

What is JavaScript?

JavaScript (JS) is a lightweight, interpreted programming language with first-class functions. Originally created to make web pages interactive, it has grown into one of the most widely used programming languages in the world — running both in browsers and on servers via Node.js.

JavaScript lets you dynamically update content, control multimedia, animate images, and respond to user actions like clicks and keyboard presses — all without reloading the page.

Example 1: Variables & Output

JavaScript uses let, const, and var to store values. console.log() prints output:

const name = "Asim Titung";
const role = "Full Stack Developer";
console.log("Name: " + name);
console.log("Role: " + role);

Output:

Name: Asim Titung
Role: Full Stack Developer

Example 2: Loops

A for loop repeats a block of code a set number of times:

const skills = ["HTML", "CSS", "JavaScript", "Node.js"];

for (let i = 0; i < skills.length; i++) {
  console.log((i + 1) + ". " + skills[i]);
}

Output:

1. HTML
2. CSS
3. JavaScript
4. Node.js

Example 3: Functions

Functions are reusable blocks of logic. Here's a live interactive example — click the button to run it:

function multiply(a, b) {
  return a * b;
}

const result = multiply(7, 6);
console.log("7 × 6 = " + result);

Example 4: DOM Manipulation

JavaScript can interact with HTML elements to change content, styles, or react to events:

document.getElementById("greeting")
  .textContent = "Hello from JavaScript!";

document.getElementById("greet-btn")
  .addEventListener("click", function() {
    alert("Button clicked by Asim!");
  });

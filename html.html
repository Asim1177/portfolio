HyperText Markup Language — the structure of every web page

What is HTML?

HTML (HyperText Markup Language) is the foundational language of the web. It uses elements and tags to define the structure and meaning of web content — headings, paragraphs, links, images, forms, and more. Every website you've ever visited is built on HTML at its core.

HTML works together with CSS (for styling) and JavaScript (for behaviour) to produce the full web experience. A browser reads an HTML document and renders it as a visual, interactive page.

Example 1: Document Structure

Every HTML page follows the same fundamental skeleton:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Asim's First Page</title>
</head>
<body>
  <h1>Hello, I'm Asim Titung</h1>
  <p>Welcome to my web page!</p>
</body>
</html>

Output:

Hello, I'm Asim Titung

Welcome to my web page!

Example 2: Links, Images & Media

HTML lets you embed clickable links and images using simple tags:

<h3>Useful Resources</h3>
<a href="https://developer.mozilla.org" target="_blank">Visit MDN Docs</a>

<h3>A Sample Image</h3>
<img src="https://via.placeholder.com/250x140" alt="Sample placeholder image">

Output:

Useful Resources

Visit MDN Docs

A Sample Image

Example 3: Lists & Tables

Organise content with unordered lists and structured tables:

<h3>My Favourite Technologies</h3>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
</ul>

<h3>Quiz Results</h3>
<table border="1">
  <tr><th>Student</th><th>Mark</th></tr>
  <tr><td>Asim</td><td>97</td></tr>
  <tr><td>Jordan</td><td>85</td></tr>
</table>

Output:

My Favourite Technologies

  • HTML5
  • CSS3
  • JavaScript

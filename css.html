Cascading Style Sheets — making the web beautiful

What is CSS?

CSS (Cascading Style Sheets) is the language used to control the visual presentation of HTML documents. It handles everything from colours, fonts and spacing to complex animations and responsive layouts. Without CSS, every website would look like a plain text document.

CSS works by selecting HTML elements and applying declarations (property-value pairs) to them. Multiple stylesheets cascade together in a defined priority order, which is where the "cascading" in the name comes from.

Example 1: Typography Styling

Control font size, colour, alignment and style with a few simple rules:

h1 {
  color: #22d3ee;
  text-align: center;
  font-size: 2rem;
}
p {
  color: #94a3b8;
  font-style: italic;
  line-height: 1.8;
}

Example 2: Box & Card Styling

CSS gives elements background colour, borders, padding and rounded corners:

.card {
  background: rgba(59,130,246, 0.1);
  border: 2px solid #3b82f6;
  padding: 18px;
  border-radius: 14px;
  width: 240px;
}

Example 3: Hover Effect

The :hover pseudo-class applies styles when the user moves their cursor over an element:

.btn {
  background: #3b82f6;
  color: #fff;
  padding: 10px 22px;
  border-radius: 8px;
  border: none;
  transition: background 0.2s;
}
.btn:hover {
  background: #1d4ed8;
}
